# Manpower-Management-Dashboard
This Power BI dashboard provides an interactive and data-driven view of employee and manpower information within the organization. It is designed to help HR teams, project managers, and leadership efficiently manage workforce allocation and optimize resource utilization.
The dashboard integrates employee data to track manpower availability, identify skill gaps, and ensure the right people are deployed to the right projects. It also highlights employees in the talent pool, enabling managers to quickly assign them to projects where additional support is needed.
A Python script is used for data processing and automation, ensuring the dashboard remains up to date and provides actionable insights.
